# Демо-версия лендинга «Строим Мир»

## Структура
В данной папке находится финальный, полностью самодостаточный файл лендинга:
- `index.html` — единый файл, содержащий всю верстку, стили и скрипты. Не требует внешних CSS/JS библиотек.

## Технические особенности (Информация для разработчиков / Claude)
Страница собрана с применением технологии **Shadow DOM**. Это сделано намеренно для обеспечения 100% изоляции стилей (CSS) каждого отдельного блока.
1. Все блоки (секции лендинга) лежат внутри тегов `<template id="tmpl-...">`.
2. Внизу документа расположен небольшой JavaScript-код, который при загрузке монтирует каждый `<template>` в соответствующий `<div id="...">` через `attachShadow({mode: 'open'})`.
3. Внутри каждого теневого дерева (Shadow Tree) лежат свои локальные `<style>`, которые никак не влияют на соседние блоки, предотвращая конфликты классов (например, глобальных стилей `body`, `.container`, `.title`).
4. Навигация (якорные ссылки в шапке и подвале) специально адаптирована для работы поверх Shadow DOM — JavaScript перехватывает клики и выполняет `scrollIntoView()` до нужного Shadow-контейнера.
5. Изображения подгружаются по прямым ссылкам (например, Unsplash).
6. Файл сохранен в кодировке **UTF-8**, чтобы корректно отображалась кириллица.

## Как развернуть на демо-стенде
Никакой сборки (Webpack, Vite, npm) не требуется. Достаточно просто положить файл `index.html` на любой статический хостинг (GitHub Pages, Netlify, Vercel, Яндекс.Облако, или обычный Nginx/Apache сервер). Страница сразу готова к просмотру.
